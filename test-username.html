Инструкция:

  1. Переименуйте файл или папку так, чтобы путь содержал "/deal/" (например: C:\deal\test-username.html)
  2. Откройте эту страницу в браузере с установленным расширением
  3. Откройте консоль (F12) и введите: autoCopyDeliveryData()
  4. Проверьте буфер обмена - должен скопироваться username
  5. Проверьте, что переключатель "Автокопирование" включен в панели расширения
  6. Проверьте localStorage: localStorage.getItem('dist_autocopy_enabled') (должно быть null или 'true')

Для теста в консоли:

// Проверить, загружена ли функция
typeof autoCopyDeliveryData

// Проверить состояние автокопирования
localStorage.getItem('dist_autocopy_enabled')

// Включить автокопирование
localStorage.setItem('dist_autocopy_enabled', 'true')

// Вызвать функцию вручную
autoCopyDeliveryData()
